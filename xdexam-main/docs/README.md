### 项目介绍

学之思开源考试系统是一款 java + vue 的前后端分离的考试系统。主要优点是开发、部署简单快捷、界面设计友好、代码结构清晰。支持web端和微信小程序，能覆盖到pc机和手机等设备。 支持多种部署方式：集成部署、前后端分离部署、docker部署。

* 码云最有价值开源项目
* 累计star数超过 5000+
* 开源在线考试系统排行首位
* 交流人数超过 4000+
